# Frontend State Review Checklist

## 分類と所有者

- [ ] 状態をUI、共有、サーバー、URL、フォーム、オフラインに分類した
- [ ] 正規ソース、所有者、保存先、寿命、削除条件を記録した
- [ ] 派生値を二重保持していない

## 更新とキャッシュ

- [ ] イベント、非同期処理、Abort、再試行、invalidateを定義した
- [ ] loading、empty、error、stale、権限、期限切れを扱った
- [ ] 楽観更新の復元と競合方針を確認した
- [ ] 不要な再レンダー、キャッシュ、メモリを測定した

## オフラインと安全性

- [ ] 未送信操作、再送、冪等性、競合、復帰を定義した
- [ ] navigator.onLineだけに依存していない
- [ ] 個人情報・秘密を不適切に永続保存していない
- [ ] Service Workerのキャッシュ範囲と失効を確認した

## 検証と運用

- [ ] リロード、履歴、SSR/水和、複数タブ、低速回線を確認した
- [ ] 正常、空、エラー、権限、期限切れ、復帰をテストした
- [ ] ロールバック、残存リスク、非対応範囲を記録した
