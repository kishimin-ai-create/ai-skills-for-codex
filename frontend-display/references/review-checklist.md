# Frontend Display Review Checklist

## 環境とレイアウト

- [ ] 対応ブラウザ、OS、端末幅、入力方式、ズーム要件を記録した
- [ ] コンテンツが破綻する幅を確認し、ブレークポイントを決めた
- [ ] 長い翻訳、拡大表示、キーボードフォーカスで情報が隠れない

## メディアとテーマ

- [ ] alt、画像形式、サイズ、遅延読み込み、著作権を確認した
- [ ] ライト／ダーク双方でコントラスト、フォーカス、状態表示を確認した
- [ ] フォント、LCP、CLS、キャッシュ、圧縮を測定した

## i18nとアクセシビリティ

- [ ] 翻訳関数、日付・数値・通貨フォーマッターを使用した
- [ ] title、favicon、aria-label、alt、loading、error文言を確認した
- [ ] キーボード、スクリーンリーダー、タッチ、色覚、reduced motionを確認した

## 検証と運用

- [ ] 代表的なブラウザ・端末・言語・テーマ・ネットワークで検証した
- [ ] 非対応範囲、測定結果、残存リスク、ロールバックを記録した
- [ ] 機能追加や無関係なリファクタリングを表示変更へ混在させていない
