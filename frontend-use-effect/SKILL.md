# useEffect

## 目的

`useEffect` を外部システムとの同期にのみ使用すること。

## 手順

### 1. 外部システムとの同期か確認する

`useEffect` を使用する前に、処理が外部システムとの同期であることを確認する。

外部システムの例:

- DOM API
- WebSocket
- サードパーティの購読処理

### 2. UI ロジックや状態遷移では使用しない

UI ロジックや状態遷移を制御するために `useEffect` を使用してはならない。

代わりに、以下を使用する。

- イベントハンドラー
- 導出された状態
- `useMemo`
- カスタムフック
