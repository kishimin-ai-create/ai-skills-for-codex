# Human Interface Guideline

## 目的

UIやUXの設計、実装、レビューに Human Interface Guideline の原則を適用すること。

## 手順

1. 対象となるUIやUXの目的と主要なユーザー操作を確認する。
2. 設計、実装、レビューの対象に関連する原則を特定する。
3. 関連する原則に従って設計または実装する。
4. 原則間で競合がある場合は、ユーザーの主要な目的と使いやすさを優先して判断する。
5. 完了前に関連する原則への違反がないか確認する。
